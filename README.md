# livetrans_prototype

## Requirements

1. [faster-whisper](https://github.com/SYSTRAN/faster-whisper)
2. [silero-vad](https://github.com/snakers4/silero-vad)
3. `openai`
4. `torch`
5. `python-telegram-bot`
6. `requests`
7. `httpx`
8. `numpy`
9. ffmpeg

## Run
1. Fill in creds & ffmpeg paths in `tg_test.py`
